Detailed Description

template<class Generator, class... Steps>
class caf::flow::op::from_generator< Generator, Steps >

Converts a Generator to an observable.

Note
Depending on the Generator, this operator may turn cold if copying the generator results in each copy emitting the exact same sequence of values. However, we should treat it as hot by default.
